# Organic diradical D-NIT2

Reproducing temperature dependence of magnetic susceptibility
from [Ziessel R. et al.](https://doi.org/10.1021/ja0305959)
using Spinner.

![D-NIT2](10.1021_ja0305959/d-nit2.png)

ESR data: $g = 2.007$, $J = -240 K$.
